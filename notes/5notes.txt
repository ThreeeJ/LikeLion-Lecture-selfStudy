# 스텝 05-02, MySQL과 Dbeaver 설치
- 참고사항 : 크롬은 네이버가 아닙니다. 마찬가지로 Dbeaver 도 MySQL 이 아닙니다. 단지 MySQL 서버에 접속하기 위한 도구입니다.

- docker를 이용하여 mysql 설치
- 비밀번호 : lldj****14 // *는 가리기용도
```
# MySQL 없을 때 띄우는 방법
cd ~ # 운영환경에서는 `cd /`

# 기존 컨테이너와 볼륨 제거
docker ps -a | grep -q mysql-1 && docker rm -f mysql-1
rm -rf dockerProjects/mysql-1

# 설정파일 만들기
mkdir -p dockerProjects/mysql-1/volumes/etc/mysql/conf.d

# 원하는 설정을 적어주세요.
echo "[mysqld]
# general_log = ON
# general_log_file = /etc/mysql/conf.d/general.log" > dockerProjects/mysql-1/volumes/etc/mysql/conf.d/my.cnf
chmod 444 dockerProjects/mysql-1/volumes/etc/mysql/conf.d/my.cnf

docker run \
    --name mysql-1 \
    -p 3306:3306 \
    -v /${PWD}/dockerProjects/mysql-1/volumes/var/lib/mysql:/var/lib/mysql \
    -v /${PWD}/dockerProjects/mysql-1/volumes/etc/mysql/conf.d:/etc/mysql/conf.d \
    -e TZ=Asia/Seoul \
    -e MYSQL_ROOT_PASSWORD=lldj****14 # *는 가리기용도
    -d \
    mysql:8.4.1
```

# 스텝 05-03, 데이터베이스와 MySQL
1. DB (Database) = 데이터베이스 = 폴더
- 폴더는 여러 정보를 묶어주는 것
- DB는 테이블들로 구성
	- DB는 테이블들을 묶어주는 것

2. MySQL(=MariaDB)과 DBMS
- MySQL은 전세계에서 가장 많이 사용되는 DBMS 입니다.
- DBMS (DataBase Management System)는 여러개의 DB를 관리하는 시스템입니다.
- MySQL은 DBMS 중에 하나, 다른 DBMS로는 오라클, MSSQL 등이 있습니다.

3. 테이블 = 표 = 엑셀
- 표 == 칼럼(열, 1줄) + 로우(행, 여러줄)
	- 컬럼(Column): == 번호, 이름, 나이, 연락처, 이메일
		- 표에서 칼럼은 한줄만 존재한다.
	- 로우(Row): == 1, 홍길동, 24, 010-1234-1234, test@test.com
		- 표에서 로우는 0 ~ N줄 존재할 수 있다.

4. SQL (Structured Query Language)
- SQL은 명령어 이다.
- SQL은 DB에게 명령을 내릴 때 사용된다.

- SQL 명령어
	<전체 데이터베이스 리스팅>
	SHOW DATABASES;

	<특정 데이터베이스 선택, 다른말로 하면 특정 폴더(DB)로 접속 명령어>
	USE \`DB이름\`;
		EX : USE `mysql`;

	<현재 접속해 있는 DB안의 모든 테이블들을 리스팅>
	SHOW TABLES;
